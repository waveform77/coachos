# CoachOS — Product Vision

> «The Operating System for Football Academies»

## Elevator Pitch

CoachOS — это операционная система для футбольной академии. Она объединяет в едином workspace тренеров, игроков, родителей и менеджмент клуба: планирование тренировок, библиотеку упражнений, оценку прогресса, аналитику нагрузок и AI-ассистента тренера.

Тренер тратит меньше времени на Excel и больше — на футбол. Родитель видит реальный прогресс ребёнка, а не сухой график.

| Сегмент | Описание | Размер |

| Любительские клубы | Взрослые команды без профессионального менеджмента | Вторичный ICP |

## Ключевая проблема

### «Клеточный хаос» в управлении академией

Сегодня типичная футбольная академия на 100–200 игроков работает так:

1. **Тренер** ведёт план тренировок в Word/Excel, присылает в Telegram-группу
2. **Посещаемость** отмечается в тетради или Google Sheets
3. **Прогресс игрока** нигде не хранится систематически — «в голове тренера»
4. **Родители** узнают о результатах на словах раз в месяц
5. **Директор академии** не видит работу тренеров объективно
6. **Нагрузка** планируется «на глазок» — травмы из-за перегрузок у детей

### Следствия
- Тренер тратит 3–5 часов в неделю на административную работу вместо тренировочного процесса
- Прогресс игрока невозможно измерить и предъявить родителям объективно
- Академия не может масштабироваться — знания «в голове» не тиражируются
- Родители не понимают, за что платят

## Решение

CoachOS предоставляет **единый workspace** с ролевыми представлениями:

### Для тренера
- Конструктор тренировок из библиотеки упражнений (вместо Word)
- Calendar с сессиями и матчами
- Оценка игроков по 5 параметрам → автоматический Player Development Index
- Coach Command Center: кто сегодня отсутствует, кто перегружен, у кого падает прогресс
- AI Coach Assistant: генерирует план тренировки под цель за 10 секунд

### Для игрока
- Личный кабинет: расписание, прогресс, цели
- Визуальный прогресс по параметрам (Radar Chart)
- Рекомендации от тренера и AI

### Для родителя
- **Parent Transparency Mode**: понятный прогресс без сухой статистики
- Посещаемость, сильные стороны, зоны роста
- Ближайшие события, отчёты

### Для администратора академии
- Обзор всего клуба: команды, тренеры, игроки
- Аналитика эффективности тренеров
- Управление доступами

## Уникальное торговое предложение (UVP)

> «Не CRM и не календарь. CoachOS — это Notion + Strava + Linear для футбольной академии: одна система, в которой тренер планирует, академия управляет, игрок растёт, а родитель видит результат.»

## Инновационные фичи

### 1. Player Development Index (PDI)
Нормированный индекс 0–100, рассчитываемый по формуле:
```
PDI = (attendance_rate × 0.20) + (avg_assessment × 0.50) + (goals_achieved_rate × 0.30)
```
Позволяет объективно сравнивать прогресс игроков и отвечать родителям «как развивается ваш ребёнок».

### 2. Smart Training Load
Алгоритм расчёта еженедельной нагрузки на основе интенсивности и длительности тренировок. Предупреждает о потенциальной перегрузке до того, как случится травма.

### 3. AI Training Planner
Mock-реализация AI-ассистента с архитектурой под подключение OpenAI/Claude. Генерирует:
- Структуру тренировки по цели (прессинг, первый пас, выносливость)
- Рекомендованные упражнения под слабые стороны игрока
- Анализ прогресса и риски

### 4. Weakness-to-Drill Engine
Если у игрока оценка по «технике» ниже 5, система автоматически предлагает упражнения категории «technique» из библиотеки клуба.

### 5. Parent Transparency Mode
Особый режим для родителей — не сухие числа, а нарратив: «Иван за последние 3 месяца улучшил показатель тактического понимания на 2 балла, посещаемость — 87%».

## Бизнес-модель (future)

- **Freemium**: до 20 игроков бесплатно
- **Academy**: $99/месяц до 100 игроков
- **Pro**: $299/месяц неограниченно + API + интеграции
- **Enterprise**: кастомный контракт для профессиональных клубов

## Почему CoachOS полезен

**Тренеру:** экономит 3–5 часов в неделю, профессиональный инструмент вместо тетради  
**Игроку:** видит свой прогресс, цели мотивируют  
**Родителю:** понимает, за что платит, видит прогресс ребёнка  
**Клубу:** масштабируемая система, объективная аналитика, профессиональный имидж
